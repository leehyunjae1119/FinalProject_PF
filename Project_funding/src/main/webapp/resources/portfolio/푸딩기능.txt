크라우드 펀딩 사이트
사용 API : 카카오 오픈API, 아임포트 결제API, 네이버 스마트에디터, 네이버 로그인, 도로명주소

UI
- JPLAYER (문서뷰어)

기능
회원유형 
- 투자자 : 프로젝트 검색, 댓글, 투자, 결제
- 파트너스 : 투자자 권한 + 프로젝트 참여
- 클라이언트 : 투자자 권한 + 프로젝트 등록
- 관리자 : 회원 관리 및 계정유형 심사

회원관리
- 로그인
- 회원가입 : 기본적으로 투자자로 회원가입되며 관리자에게 요청 및 승인에 의해 파트너 또는 클라이언트 계정으로 전환
- 회원정보 : 회원 정보 조회 수정 삭제 기능 및 포트폴리오 등록(파일 업로드)
- 계정유형 전환 신청 : 프로젝트 또는 이력서 심사 신청 (파일업로드)
- 마이페이지 : 내 활동내역, 개인정보 조회, 포트폴리오or프로젝트 등록및 수정
- 회원검색 : 파트너스, 클라이언트 검색, 활동내역 조회, 프로젝트 참여 권유기능
- 관리자페이지 : 회원관리 관리자 기능, 계정유형 전환 심사

투자
- 투자 : 투자자가 포인트를 충전하여 프로젝트에 투자
- 반환금 : 목표 투자금 달성 미달시 투자자에게 포인트 반환
- 투자 정보 : 투자자가 투자한 프로잭트의 정보 진행결과 확인

게시판
- 카테고리(자바, 안드로이드, 파이썬 등등)
- CRUD
- AJAX를 통한 상세정보 필터링
- PPT 프리뷰 출력
- 페이징
- 검색
- 이용방법

커뮤니티
- 카카오 오픈채팅방 초대 : 파트너스 유저가 프로젝트에 참여하면 프로젝트 담당자가 있는 카카오 오픈채팅방으로 들어갈 수 있음
- 메시지 : 파트너스 유저와 프로젝트 담당자가 서로 메세지를 주고 받는다.

결제
- 포인트 충전/사용 : 결제 시스템을 이용
- 포인트 환불 

시각화
- d3.js 를 통한 데이터 시각화
